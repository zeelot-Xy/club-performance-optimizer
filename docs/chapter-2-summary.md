# Chapter 2 Summary

## Literature Direction
Chapter 2 should position the project within:
- sports analytics and football decision support
- player performance evaluation methods
- rule-based expert systems
- machine learning for structured sports data
- explainable AI in practical decision environments

## Key Argument
Many sports analytics systems focus on broad enterprise problems, heavy live data, or complex predictive pipelines. This project intentionally addresses a narrower and more defendable gap: explainable weekly recommendation support for one club using maintainable structured data.

## Review Position
- rule-based systems are strong for interpretability and traceability
- machine learning can add value, but may reduce immediate transparency if overused
- therefore, the project adopts a hybrid approach:
  - rule engine as primary recommendation logic
  - Random Forest support layer as optional enhancement

## Review Gap Statement
There is academic value in designing a finishable, explainable, single-club decision-support system that prioritizes transparency and implementation realism over excessive scope.
