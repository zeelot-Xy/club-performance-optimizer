# Objectives

## General Objective
To develop an explainable AI-based web system that analyzes football player performance and recommends an optimal weekly formation and starting lineup for a single club.

## Specific Objectives
- Design a database for player and weekly performance records
- Build secure backend APIs for squad and weekly data management
- Develop a rule-based scoring engine for weekly player evaluation
- Recommend a formation and lineup using explainable logic
- Build a visual dashboard for coaches
- Integrate an optional ML model for predictive enhancement
- Evaluate correctness, usability, and explainability of the system
