# Technical Decision Record

## Frontend
React + TypeScript chosen for component-based UI, maintainability, and strong ecosystem support.

## Backend
Express + TypeScript chosen for predictable API development and shared type-safe engineering practices.

## Database
PostgreSQL chosen for relational consistency and Prisma integration.

## AI Service
FastAPI chosen because Python tooling is ideal for scikit-learn experimentation and clear separation of ML concerns.

## Explainability Strategy
Rule-based scoring is the primary intelligence layer because it is transparent, testable, and easy to defend academically. Machine learning is an enhancement, not the foundation.
