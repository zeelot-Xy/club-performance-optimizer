# Chapter 5 Summary

## Testing and Discussion
Chapter 5 should synthesize the testing evidence and discuss what the project achieved relative to its stated objectives.

## Discussion Themes
- the constrained scope improved implementation quality
- explainability remained the primary system strength
- the hybrid data strategy was suitable for an academic prototype
- the ML layer added value without replacing transparent logic

## Limitations
- single-club only
- no real-time or sensor-driven inputs
- weekly condition data is partially synthetic
- ML layer is supplementary and not fully production-calibrated

## Conclusion Direction
The project successfully delivered a realistic web-based football decision-support platform that analyzes player performance and recommends a weekly formation and lineup in an explainable and maintainable manner.

## Future Work Suggestions
- add weekly performance entry UI for full admin workflow completion
- expand dashboard analytics
